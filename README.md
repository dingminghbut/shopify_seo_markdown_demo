# Shopify Homepage Health Checker

本地运行的 Shopify 店铺首页健康检测工具。

## 前置要求

- Node.js 18+

## 安装

```bash
npm install
```

## 启动

```bash
npm start
```

## 访问

浏览器打开 http://localhost:3000

## 使用说明

1. 输入 Shopify 店铺 URL（例如：`https://example.myshopify.com`）
2. 点击「开始检测」按钮
3. 查看健康检测报告
4. 可选择下载 PDF 报告

## 技术栈

- Node.js
- Express
- Cheerio
- Axios
- marked.js
